# 🛠️ FONTS FORGE by Ethernium

![Ethernium 見本シート](../ethernium_sheet_hq.png)

手描きまたはグリッドベースのラスター見本シート（PNG）から**カスタムベクターフォントを設計、コンパイル、可視化する**ための、プロフェッショナルで最先端のツールキットです。

法科学レベルの精密さとサイバーパンク美学で構築されています。

---

## ✨ 特徴

- **汎用ラスター→ベクター変換パイプライン**: グリッドベースの見本を完全な文字バウンディングボックスに自動分割し、グリフの輪郭を抽出してベクター形式に変換します。
- **幾何学的スナッピングとスムージング**: 設定可能な角度スナッピング（45° / 90°）とモルフォロジカルエッジフィルターにより、鮮明なディテールを保ちながらジャギーを除去します。
- **ベジェ曲線フィッティング**: 偏向角解析を使用した二次/三次ベジェの自動分類により、プロフェッショナルグレードの曲線を実現します。
- **デュアルレイヤー対称エンジン**: ベクターレベルとピクセルレベルの両方で数学的に完全な対称性を実現するために、輪郭を左右に反転します。
- **フォレンジック電子透かし**: 著作者証明のためのLSBベースのステガノグラフィック座標埋め込み。
- **プロフェッショナルOpenTypeテーブル**: 堅牢なOS/2垂直メトリクス、`gasp`画面レンダリングヒンティング、著作権レコード、レガシーカーニングマップ。
- **マルチフォーマット出力**: 単一ビルドで`.ttf`、`.woff`、`.woff2`を生成します。

## 🚀 4ステップでカスタムフォントを作成

### ステップ1: 見本シートを描く
フォントのグリフを1枚のPNG画像に描画または構成します。文字を左から右に行ごとに配置します。

### ステップ2: プロジェクトを設定する
`configs/template.json`を`configs/my_font.json`にコピーして、以下を定義します：
- `"sheet"`: PNGファイル名
- `"font"`: 著作権、ファミリー名、スタイル名
- `"rows"`: Y座標と各行の文字リスト（順序付き）

> 💡 **自動キャリブレーション**: `python tools/calibrate_sheet.py my_sheet.png`を実行して、Y境界を自動的に検出します。

### ステップ3: コンパイル
```bash
pip install -r requirements.txt
python -m font_forge configs/my_font.json
```

### ステップ4: プレビューとデプロイ
ブラウザで`preview_font.html`を開いて、グリフグリッドの検査、サイズのテスト、埋め込みコードの取得を行います。

## 🔬 開発者ツール

| スクリプト | 目的 |
|--------|--------|
| `tools/calibrate_sheet.py` | 自動Y境界スキャンとバンド提案 |
| `tools/debug_rows.py` | 座標スライスの視覚的検証オーバーレイ |
| `tools/audit_font.py` | コンパイル済み垂直境界とグリフ範囲の整合性検証 |
| `tools/validate_font.py` | OpenType仕様監査（合格/警告/不合格レポート） |
| `tools/font_to_ascii.py` | テキストを高解像度ターミナルASCIIバナーに変換 |
| `tools/export_atlas.py` | コンパイル済みTTFからビジュアルグリフアトラスを生成 |

# 🛠️ FONTS FORGE от Ethernium

![Образец шрифта Ethernium](../ethernium_sheet_hq.png)

Профессиональный, современный инструментарий для **проектирования, компиляции и визуализации пользовательских векторных шрифтов** на основе простых нарисованных от руки или сеточных растровых образцов (PNG).

Создан с криминалистической точностью и эстетикой киберпанка.

---

## ✨ Возможности

- **Универсальный конвейер растр-в-вектор**: Автоматически разделяет сеточные образцы на точные ограничивающие рамки символов, извлекает контуры глифов и преобразует их в векторные форматы.
- **Геометрическая привязка и сглаживание**: Настраиваемая привязка углов (45° / 90°) и морфологические краевые фильтры для устранения ступенчатости при сохранении чётких деталей.
- **Аппроксимация кривыми Безье**: Автоматическая классификация квадратичных/кубических кривых Безье с использованием анализа угла отклонения для профессиональных кривых.
- **Двухуровневый движок симметрии**: Зеркальное отражение контуров слева направо для математически идеальной симметрии на уровне вектора и пикселей.
- **Криминалистические водяные знаки**: Стеганографическое встраивание координат на основе LSB для подтверждения авторства.
- **Профессиональные таблицы OpenType**: Надёжные вертикальные метрики OS/2, хинтинг экранного рендеринга `gasp`, записи об авторских правах и устаревшие карты кернинга.
- **Многоформатный вывод**: Генерация `.ttf`, `.woff` и `.woff2` за одну сборку.

## 🚀 Создайте свой шрифт за 4 шага

### Шаг 1: Нарисуйте образец
Нарисуйте или создайте глифы вашего шрифта в одном PNG-изображении. Расположите символы слева направо по строкам.

### Шаг 2: Настройте проект
Скопируйте `configs/template.json` в `configs/my_font.json` и определите:
- `"sheet"`: Имя вашего PNG-файла
- `"font"`: Авторские права, название семейства, название стиля
- `"rows"`: Y-координаты и упорядоченный список символов для каждой строки

> 💡 **Автокалибровка**: Выполните `python tools/calibrate_sheet.py my_sheet.png` для автоматического определения Y-границ.

### Шаг 3: Компиляция
```bash
pip install -r requirements.txt
python -m font_forge configs/my_font.json
```

### Шаг 4: Просмотр и развёртывание
Откройте `preview_font.html` в браузере для проверки сетки глифов, тестирования размеров и получения кода встраивания.

## 🔬 Инструменты разработчика

| Скрипт | Назначение |
|--------|--------|
| `tools/calibrate_sheet.py` | Автоматическое сканирование Y-границ и предложение полос |
| `tools/debug_rows.py` | Визуальная проверка наложением координатных срезов |
| `tools/audit_font.py` | Проверка целостности скомпилированных вертикальных границ и диапазонов глифов |
| `tools/validate_font.py` | Аудитор спецификации OpenType (отчёты: успех/предупреждение/ошибка) |
| `tools/font_to_ascii.py` | Преобразование текста в высокоразрешённые ASCII-баннеры для терминала |
| `tools/export_atlas.py` | Генерация визуального атласа глифов из скомпилированного TTF |
